Quick start:

build strads lib
```
cd strads-root-dir
make
```

build application 
```
cd ./apps/your-app-dir
Make in app/your app 
```

example for LDA 
```
cd strads-root-dir 
make 
cd ./apps/lda_release 
make 
./single.py
```
you need to modify single.py to set path to the input file
